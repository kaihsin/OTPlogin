# Simple OTP login for Taiwania HPC

## Prequisition:

    * sshpass
    * pyotp
    * python v3

## Setting:

    * OTPKey: Please fillin the OTPKey
    * Host  : The IP address (or domain-name) 
    * User  : The user ID
    
## Usage:
    
    $python login.py
